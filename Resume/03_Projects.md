## 1. My Robi *(Self-Care Telecom Application)*
* **Role:** Mobile Developer (Android Native)
* **Platform:** Android Native (Kotlin, XML, MVVM)
* **API Client:** Retrofit, OkHttp
* **Overview:** A leading self-care telecom application in Bangladesh with millions of active users. It allows customers to manage accounts, recharge balances, buy mobile internet packs, and activate customized telecom bundles.
* **My Responsibilities:** Developed and updated existing features with new UI layouts, resolved production-level application issues, and integrated REST APIs.
* **Technical Challenge:** App lag and memory overhead due to deeply nested layouts, resolved by flattening view hierarchies with ConstraintLayout.
* **Business Impact:** Improved application stability and enhanced the self-care experience, contributing to higher user satisfaction and retention.
